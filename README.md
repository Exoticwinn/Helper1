README: Telegram‑бот с утилитами
Это простой Telegram‑бот, реализующий набор развлекательных и утилитарных команд.

Функционал
Бот поддерживает следующие команды:

/start, /hello — приветственное сообщение.

/heh [число] — повторяет «he» заданное число раз (по умолчанию 5).

/bye — прощальное сообщение.

/password — генерирует случайный пароль из 8 символов.

/emodji — отправляет случайный эмодзи из предустановленного списка.

/coin — «подбрасывает монетку» и возвращает «ОРЕЛ» или «РЕШКА».

/fact — присылает случайный интересный факт из базы.

/countdown [число] — запускает обратный отсчёт от заданного числа до 0 (с ограничением: 0 ≤ число ≤ 30).

Если бот получает любое сообщение без команды, он повторяет его (эхо‑режим).

Структура проекта
Проект состоит из двух основных файлов:

logic.py — содержит логику работы бота:

gen_pass(pass_length) — генерация пароля заданной длины.

gen_emodji() — выбор случайного эмодзи.

flip_coin() — подбрасывание монетки.

get_random_fact() — выбор случайного факта.

count_down(n) — формирование строки с обратным отсчётом через стрелку (например, 5 → 4 → 3 → 2 → 1 → 0).

main.py — точка входа, настройка бота и обработка команд:

Инициализация бота через telebot.TeleBot.

Обработчики команд (@bot.message_handler(commands=...)).

Эхо‑обработчик для всех прочих сообщений.

Требования
Для работы бота необходимы:

Python 3.x

Библиотека pyTelegramBotAPI (устанавливается через pip install pyTelegramBotAPI)

Файл config.py с переменной TOKEN (токен вашего Telegram‑бота)

Настройка и запуск
Установите зависимости:

bash
pip install pyTelegramBotAPI
Создайте файл config.py и добавьте в него токен бота:

python
TOKEN = "ваш_токен_здесь"
Запустите бот:

bash
python main.py
Ограничения и особенности
Команда /countdown принимает только целые неотрицательные числа от 0 до 30.

Пароль всегда генерируется длиной 8 символов (можно модифицировать в коде).

Список фактов и эмодзи фиксирован; для расширения нужно отредактировать соответствующие списки в logic.py.

Обратный отсчёт в /countdown отправляется отдельными сообщениями с задержкой 1 сек. между ними.
